# Todo-app

## Setup

```sh
yarn install
```

### Mac

1. Download and install Xcode
2. Install Pods

```sh
# Install pods
cd ios && pod install
```

### Android

- Download and install Android Studio
- Ensure you have an emulator with API >= 21
## Launching the App

```sh
# Start the packager
yarn start

# In another terminal 
# Start the app for iOS or Android
yarn ios
yarn android
```
